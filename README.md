# vocalo

A minimalist C++ language learning app with a web UI.

## Build
```bash
g++ -O3 main.cpp -o vocalo -lpthread
./vocalo
```

## Usage
1. Run `./vocalo`.
2. Open `http://localhost:8080`.

## Features
- Custom deck support (JSON).
- Session tracking and scoring.
- Web-based flashcard/quiz interface.
